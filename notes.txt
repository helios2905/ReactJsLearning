to be installed:
npm install jquery --save
npm install --save-dev coffee-loader
npm install --save-dev html-webpack-plugin
npm install --save-dev babel-core babel-loader babel-preset-react   

to run a command test:
npm run test (after setting test in module-loaders)

npm install webpack
to run webpack:
webpack -w (to watch files)
webpack -p (run + minify code)
